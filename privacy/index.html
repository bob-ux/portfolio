Политика конфиденциальности

Информация об обработке данных на сайте forpikbob.ru

1. Общие положения

Настоящая политика конфиденциальности определяет порядок обработки информации, получаемой при использовании сайта forpikbob.ru.

2. Какие данные могут обрабатываться

При посещении сайта могут автоматически обрабатываться технические данные, включая IP-адрес, сведения о браузере, устройстве, cookies, а также данные о действиях пользователя на сайте с использованием сервисов аналитики.

3. Цели обработки

Данные используются для анализа посещаемости сайта, улучшения структуры, содержания и пользовательского опыта.

4. Сервисы аналитики

На сайте используется Google Analytics, который может собирать обезличенные данные о посещении страниц, взаимодействии с контентом и технических параметрах устройства.

5. Передача данных третьим лицам

Данные могут обрабатываться сторонними сервисами аналитики в рамках их политики конфиденциальности и условий использования.

6. Контакты

По вопросам, связанным с использованием сайта, можно связаться по контактам, указанным на сайте.

7. Изменения политики

Политика конфиденциальности может обновляться. Актуальная версия всегда размещается на этой странице.
